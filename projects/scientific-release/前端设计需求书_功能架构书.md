## 一、项目概述

### 1.1 项目定位

**科学放生小程序**是一个面向高端用户群体的放生指导平台，倡导**科学、合规、生态友好**的放生理念。

**核心价值：**
- 🐟 科学指导：15+ 种常见放生物种，科学放生指南
- 📜 祈福证书：动态电子证书，烫金视觉效果
- 📅 佛历日历：斋戒日标注，宜忌查询
- 🏆 功德排行：善行记录，激励分享
- 🎨 现代东方禅意：高端设计风格

### 1.2 目标用户

- **主要用户：** 35-65 岁，有放生习惯的佛教信众
- **次要用户：** 对传统文化、生态保护感兴趣的人群
- **用户特征：** 注重品质、追求精神层面满足、有一定消费能力

### 1.3 技术栈

| 层级 | 技术 | 版本 |
|------|------|------|
| 前端 | 微信小程序（原生） | - |
| 语言 | WXML + WXSS + JavaScript | ES6+ |
| 后端 | Node.js + Express.js | 24.13.0 |
| 数据库 | MySQL | 8.0.44 |
| ORM | Sequelize | Latest |
| 服务器 | Linux (Alibaba Cloud) | - |
| 反向代理 | Nginx | 1.20.1 |

### 5.2 核心工具函数

#### API 封装 (utils/api.js)
```javascript
const BASE_URL = 'https://sf.dexoconnect.com/api';

function request(options) {
  return new Promise((resolve, reject) => {
    wx.request({
      url: BASE_URL + options.url,
      method: options.method || 'GET',
      data: options.data || {},
      header: { 'Content-Type': 'application/json' },
      success: (res) => {
        if (res.statusCode === 200 && res.data.success) {
          resolve(res.data);
        } else {
          wx.showToast({ title: res.data.message || '请求失败', icon: 'none' });
          reject(res.data);
        }
      },
      fail: (err) => {
        wx.showToast({ title: '网络错误', icon: 'none' });
        reject(err);
      }
    });
  });
}
```
