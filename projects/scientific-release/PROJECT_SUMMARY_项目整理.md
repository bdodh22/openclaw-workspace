## 🎨 四、设计规范 - 高端放生平台

### 4.1 风格定位
**现代东方禅意 (Modern Zen)**

**核心理念：** "空灵"与"秩序"
- 大量留白营造呼吸感
- 精致排版体现尊贵感

**视觉关键词：** 静谧、通透、温润、流动

### 4.2 颜色系统
| 颜色名称 | 色值 | 用途 |
|----------|------|------|
| **黛绿 (Deep Moss Green)** | `#4A5D4E` | 主色调、按钮、强调 |
| **青瓷色 (Celadon)** | `#8FA89E` | 辅助色块、背景 |
| **暖沙金 (Warm Sand Gold)** | `#C9B037` | 强调按钮、图标高光、会员标识 |
| **云雾白 (Mist White)** | `#F5F5F0` | 背景色 |
| **米宣纸色 (Rice Paper)** | `#EFEEE9` | 卡片背景 |
| **深岩灰 (Slate Grey)** | `#4A4A4A` | 文字颜色 |

**配色策略：** 70% 留白/米色背景 + 25% 黛绿/青瓷色块 + 5% 暖沙金点缀

### 4.3 设计趋势应用
1. **液态玻璃拟态 (Liquid Glass)** - 磨砂玻璃效果 + 动态光泽流动
2. **新中式排版** - 宋体/楷体标题 + 无衬线正文
3. **Bento Grid 布局** - 非对称圆角卡片排列
4. **沉浸式首屏** - 全屏动态水墨/流体视频背景

### 4.4 页面优化清单
| 页面 | 当前状态 | 目标设计 | 优先级 |
|------|----------|----------|--------|
| 首页 | 基础卡片布局 | 全屏意境视频 + Bento 网格 + 禅语打字机 | ⭐⭐⭐ |
| 物种列表 | 电商货架式 | 生命图鉴式 + 手绘插画 + 悬浮卡片 | ⭐⭐⭐ |
| 证书页 | 静态页面 | 动态 H5 + 烫金效果 + 数字藏品感 | ⭐⭐⭐ |
| 排行榜 | 基础榜单 | 古典卷轴样式 + 前三名特殊标识 | ⭐⭐ |
| 交互细节 | 标准跳转 | 缓动动画 + 仪式感微动画 + 触觉/听觉反馈 | ⭐ |
