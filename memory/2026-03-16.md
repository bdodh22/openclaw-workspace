# 2026-03-16 - 科学放生小程序开发日志

## 今日完成

### 🎯 Phase 2 核心功能 - 物种数据库

**1. 创建物种数据集** (`backend/data/species-data.json`)
- 包含 15 个适合放生的中国本土物种
- 分类：鱼类 (9)、鸟类 (3)、龟类 (1)、其他 (2)
- 每个物种包含：中文名、学名、栖息地、放生季节、难度、价格、功德寓意、注意事项

**物种清单：**
- 鱼类：鲫鱼、鲤鱼、泥鳅、黑鱼、黄鳝、草鱼、青鱼、鲢鱼、鳙鱼
- 鸟类：鸽子、麻雀、斑鸠
- 龟类：乌龟（中华草龟）
- 其他：田螺、青蛙

**2. 数据库模型** (`backend/src/models/Species.js`)
- Sequelize ORM 模型
- 支持分类筛选、推荐物种、搜索功能
- 字段包含完整物种信息和放生指导

**3. API 路由** (`backend/src/routes/species.js`)
- `GET /api/species` - 获取物种列表（支持 category/recommended/search 筛选）
- `GET /api/species/:id` - 获取单个物种详情
- `GET /api/species/categories/list` - 获取分类列表
- `GET /api/species/random` - 随机推荐物种（每日推荐功能）

**4. 数据库初始化脚本** (`backend/scripts/init-db.js`)
- 自动同步表结构
- 导入物种数据
- 使用方法：`node scripts/init-db.js`

**5. SQL 迁移文件** (`backend/migrations/001-create-species-table.sql`)
- 完整的表结构定义
- 包含索引优化

## 设计决策

1. **数据来源**：基于中国放生协会资料整理，确保物种为本土种，避免外来物种入侵
2. **推荐机制**：每个物种标记 `isRecommended`，黑鱼等肉食性鱼类标记为不推荐（需特殊环境）
3. **价格单位**：鱼类按"斤"，鸟类/龟类按"只"
4. **保护级别**：标注无危/易危/濒危，提醒用户注意保护动物
