## ⚠️ Git Clone 路径格式（重要！）

在 Windows 使用 Git Bash 时，路径格式必须使用：
- ✅ **正斜杠**: `E:/wechatsoft/openclaw-workspace`
- ✅ **双反斜杠**: `E:\\wechatsoft\\openclaw-workspace`
- ❌ **单反斜杠**: `E:\wechatsoft\openclaw-workspace` (会导致路径拼接错误)

**错误示例**: `git clone xxx E:\wechatsoft\openclaw-workspace` → 生成 `wechatsoftopenclaw-workspace` 文件夹

**正确命令**:
```bash
git clone https://github.com/bdodh22/openclaw-workspace.git E:/wechatsoft/openclaw-workspace
```
