放生意蕴 Significance

红耳龟，虽为常见物种，但在科学放生中承载着生态循环的重要环节。放生不仅是生命的延续，更是对自然秩序的敬畏。通过科学观测与合规放流，我们不仅救赎了生命，更在参与构建一个更加平衡的淡水生态系统。每一份善念，都如同投向湖面的石子，激起生态保护的涟漪。

放生严选 Precautions

warning

严禁跨生境投放

红耳龟极具侵略性，切勿将其放入自然原生保护区或封闭的小型生态池，以免破坏本地物种平衡。

health_and_safety

健康状态检查

观察背甲是否坚硬，眼睛是否清亮，确认无腐皮、无肺炎征兆方可进行放流。

location_on

指定放生点

请前往本程序地图标注的“政府认证科学放生点”进行操作，确保水质达标且有专业监测。
